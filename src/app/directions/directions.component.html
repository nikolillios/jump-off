Description

  • Objective: Advance your player off the end of the number line.
  • The value you spin corresponds to the number of steps you must jump forward or backwards.
  • To move your player, click the number on the numberline that corresponds to your destination.
  • If you must jump to a number less than zero, select the '< 0' tile.
  • If you must jump to a number greater than the numberline, select the 'Jump Off!' tile.
